# re-geo

react based geocities components library


## how to use

```sh
npm i re-geo

yarn add re-geo
```

```jsx
import * as React from "react";
import { Button } from "re-geo";
import "./styles.css";

export default function App() {
  return (
    <div className="App">
      <h1>Hello CodeSandbox</h1>
      <h2>Start editing to see some magic happen!</h2>
      <Button>a</Button>
    </div>
  );
}
```
